# Tier 45: WebSearch — domain filtering, JSON output, caching

**Дата**: 2026-04-28
**Статус**: ✅ Реализовано
**Цель**: Улучшить `web_search` до уровня Claude Code WebSearch — доменная фильтрация, структурированный JSON, in-memory кеш.

## Зачем

Без этих улучшений `web_search` возвращал plain text (непарсибельный для LLM), не умел фильтровать по доменам и не кешировал результаты. Каждый повторный запрос парсил DuckDuckGo заново.

## Архитектура

```
tool_web_search(params)
  ├── query: str
  ├── max_results: int (1-10, default 5)
  ├── allowed_domains: string[]? — только эти домены
  └── blocked_domains: string[]? — исключить домены

  Key → _web_cache[key] = {ts, results}
  │  └── TTL 300s (5 min), stale prune on each search

  _filter_domains(results, allowed, blocked)
    ├── allowed → filter by urlparse().netloc
    └── blocked → exclude matching

  Response → {"cached": bool, "total": int, "results": [{title, snippet, url}]}
```

### Response format (старый vs новый)

**Было** (plain text):
```
1. Python 3.13 Release
   Description...
   https://...
```

**Стало** (JSON):
```json
{
  "cached": false,
  "total": 3,
  "results": [
    {"title": "...", "snippet": "...", "url": "..."}
  ]
}
```

### Domain filtering

```python
def _filter_domains(results, allowed, blocked):
    filtered = results
    if allowed:
        filtered = [r for r in filtered if any(d in urlparse(r["url"]).netloc for d in allowed)]
    if blocked:
        filtered = [r for r in filtered if not any(d in urlparse(r["url"]).netloc for d in blocked)]
    return filtered
```

### Cache

- In-memory dict, TTL 300 секунд (5 минут)
- Cache key: `query.lower().strip()`
- Stale entries автоматически удаляются при каждом запросе (lazy eviction)
- Ограничения: безлимитный размер (в разумных пределах для LLM-агента)

## Тестирование

```bash
# Basic search (JSON output)
curl -X POST http://host:8766/tools/web-search \
  -d '{"query": "python 3.13", "max_results": 2}'
# → {"cached": false, "total": 2, "results": [{"title":..., "snippet":..., "url":...}]}

# Domain filtering
curl -X POST http://host:8766/tools/web-search \
  -d '{"query": "python news", "max_results": 5, "allowed_domains": ["python.org"]}'
# → {"total": 1, "results": [{"url": "https://www.python.org/blogs/"}]}

# Blocked domains + cache hit
curl -X POST http://host:8766/tools/web-search \
  -d '{"query": "python 3.13", "max_results": 3, "blocked_domains": ["wikipedia.org"]}'
# → {"cached": true, "total": 3, "results": [...]}  # без wikipedia.org
```
