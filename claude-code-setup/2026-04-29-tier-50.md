# Tier 50: DeepSeek Agent gap features (opt-in, zero-risk)

**Дата**: 2026-04-29
**Статус**: ✅ Развёрнуто на сервере n (8766), syntax valid, container restarted
**Цель**: Закрыть гэп между Claude Code CLI и DeepSeek Agent на сервере n — 6 opt-in фич, каждая отключена по умолчанию (zero-risk).

## 6 gap-фич (все opt-in через request body)

| # | Фича | Параметр | Default | Суть |
|---|------|----------|---------|------|
| 1 | **Budget controls** | `max_budget_usd` | 0.0 (unlimited) | Трекинг стоимости API (DeepSeek flash $0.30/$1.20, pro $4/$12 per 1M tok) + остановка при превышении |
| 2 | **Tool allow/deny** | `allowed_tools`, `blocked_tools` | None (нет фильтрации) | Фильтрация TOOL_DEFS по именам — какие инструменты разрешены/запрещены для запроса |
| 3 | **Effort levels** | `effort` | "" → medium=10 итераций | `low`=3, `medium`=10, `high`=25 итераций tool-calling loop |
| 4 | **JSON mode** | `response_format` | None | Проброс `{"type": "json_object"}` в DeepSeek API body |
| 5 | **Skip compaction** | `skip_compact` | False | Отключение context compaction per-request (для отладки) |
| 6 | **Max context tokens** | `max_context_tokens` | 0 (default) | Переопределение максимального размера контекста |
