# Tier 19: MCP Client — DeepSeek Agent подключается к внешним серверам

**Дата**: 2026-04-28
**Цель**: DeepSeek Agent умеет подключаться к MCP-серверам — GitHub, filesystem, browser, DB.

## Зачем

Claude Code подключается к MCP-серверам через `~/.claude/mcp.json`. Без этого DeepSeek Agent не может:
- Работать с GitHub (PR, issues, коммиты)
- Использовать Playwright/браузер
- Делать запросы к БД через MCP
- Подключать кастомные инструменты

## Что сделать

### 1. MCP Client (stdio + SSE)

Реализация JSON-RPC 2.0 stdlib-only:
- `connect_stdio(command, args)` — spawn процесса, stdin/stdout JSON-RPC
- `connect_sse(url)` — SSE через HTTP
- `call(method, params)` — вызов инструмента
- `list_tools()` — получение списка инструментов

```python
class MCPClient:
    def connect_stdio(self, command, args): ...
    def connect_sse(self, url): ...
    async def call(self, method, params): ...
    async def list_tools(self): ...
    def disconnect(self): ...
```

### 2. Интеграция с TOOL_DISPATCH

При запуске читать `~/.claude/mcp.json`, подключаться к серверам, регистрировать их инструменты в TOOL_DISPATCH:

```python
# При старте
for name, cfg in mcp_config.items():
    client = MCPClient()
    client.connect_stdio(cfg["command"], cfg["args"])
    tools = client.list_tools()
    for tool in tools:
        MCP_TOOLS[name] = (client, tool)
```

DeepSeek вызывает `mcp__<server>__<tool>` через TOOL_DISPATCH.

### 3. Какие MCP-серверы подключить

| Сервер | Назначение | Команда |
|--------|-----------|---------|
| GitHub | PR/issues/commits | `npx @anthropic/mcp-server-github` |
| Filesystem | безопасный FS доступ | `npx @modelcontextprotocol/server-filesystem` |
| Playwright | браузер | `npx @playwright/mcp` |
| PostgreSQL | запросы к БД | `npx @anthropic/mcp-server-postgres` |

## Зависимости

Минимум: stdlib-only (json, subprocess, http, ssl).
Для npx-серверов: Node.js в контейнере (или на хосте).
