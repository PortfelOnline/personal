# Tier 36: Session Persistence

**Дата**: 2026-04-28
**Статус**: ✅ Реализовано
**Цель**: Сохранять сессии DeepSeek Agent на диск, чтобы они переживали перезапуск Docker-контейнера.

## Зачем

Сессии хранились только в памяти (`self._sessions = {}`). При `docker compose down && up` все диалоги терялись. Добавляем save/load в SessionManager и 6 точек автосохранения.

## Что сделано

### 1. `SessionManager.save(path)` + `SessionManager.load(path)`

**Файл**: `/root/deepseek-agent/session_manager.py`

```python
def save(self, path):
    """Save all sessions to JSON file."""
    data = {}
    for sid, session in self._sessions.items():
        data[sid] = {
            "id": session.id, "created": session.created, "updated": session.updated,
            "system": session.system, "messages": session.messages, "compacted": session.compacted,
        }
    tmp = path + ".tmp"
    with open(tmp, "w") as f:
        json.dump(data, f, ensure_ascii=False, indent=2)
    os.replace(tmp, path)  # atomic — не повредит файл при сбое

def load(self, path):
    """Load sessions from JSON file. Returns count of restored sessions."""
    if not os.path.exists(path):
        return 0
    # direct Session() constructor, no session.create() to avoid side effects
```

- Атомарная запись через `.tmp` + `os.replace()` — при краше посередине записи файл остаётся целым
- `ensure_ascii=False` — кириллица сохраняется читаемо
- Полная сериализация: все поля Session (даже `compacted`)

### 3. Загрузка при старте

```python
_session_dir = os.path.join(os.path.dirname(os.path.abspath(__file__)), "sessions")
os.makedirs(_session_dir, exist_ok=True)
SESSION_FILE = os.path.join(_session_dir, "sessions.json")
_session_loaded_cnt = sessions.load(SESSION_FILE)
if _session_loaded_cnt:
    print(f"Restored {_session_loaded_cnt} session(s) from disk")
```

- `os.makedirs(exist_ok=True)` — гарантирует что директория есть
- `load()` возвращает 0 если файла нет — первый запуск без ошибок
- `print()` вместо `log()` — на уровне модуля `log()` ещё не определён

### 4. Docker volume

**Файл**: `/root/deepseek-agent/docker-compose.yml`

```yaml
- ./agent.py:/app/agent.py
- ./sessions/:/app/sessions/
```

- Bind mount: `./sessions/` на хосте → `/app/sessions/` в контейнере
- Переживает `docker compose down && docker compose up -d --force-recreate`

## Проблемы и их решение

| Проблема | Симптом | Решение |
|----------|---------|---------|
| `"""` конфликт в patch-скрипте | SyntaxError | `'''` для outer строк, `"""` для inner docstrings |
| `str.replace()` молча не находит | 6 change count, 0 изменений | `tier36_fix_agent.py` с точным содержанием agent.py |
| `log()` не определён на уровне модуля | NameError в Docker log | Замена на `print()` |
| DeepSeek API 402 Payment Required | Чат не работает | Внешняя проблема — API credits кончились |
