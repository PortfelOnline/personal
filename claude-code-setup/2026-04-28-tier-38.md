# Tier 38: Memory Store

**Дата**: 2026-04-28
**Статус**: ✅ Реализовано
**Цель**: Аналог `memory/` из Claude Code — сохранять факты, предпочтения, контекст проектов между сессиями.

## Зачем

Агент не помнил ничего между рестартами контейнера. Session Persistence (Tier 36) сохраняет историю чатов, но нет структурированного хранилища для фактов (стек проекта, API-ключи, предпочтения). Memory Store даёт key-value persistence с prefix-поиском.

## Что сделано

### 1. Хранилище

**Файл**: `/app/sessions/memory.json`

- Загружается при старте агента
- Атомарная запись через `.tmp` + `os.replace()`
- `ensure_ascii=False` — кириллица читаема

### 2. 4 инструмента

| Инструмент | Параметры | Описание |
|-----------|-----------|----------|
| `memory_set` | `key`, `value` | Сохранить значение (перезаписывает существующее) |
| `memory_get` | `key` | Получить значение по точному ключу |
| `memory_search` | `prefix` | Поиск по префиксу ключа (все совпадения) |
| `memory_delete` | `key` | Удалить запись |

### 3. Конвенция ключей

Иерархические ключи с `/` для организации:

```
project/personal/stack
project/kadmap/db_host
user/preference/theme
agent/learned_pattern/x
```

### 4. Регистрация (89 tools total)

- `TOOL_DEFS` — 4 additional definitions
- `TOOL_DISPATCH` — 4 dispatch entries
- `TOOL_URL_MAP` — 4 HTTP endpoints

### 5. Version bump

- `DeepSeek Agent v2 (Tier 19)` → `DeepSeek Agent v3 (Tier 38)`

## API

```bash
# Set
curl -X POST http://host:8766/tools/memory-set \
  -d '{"key": "project/myapp/stack", "value": "Python, FastAPI, Postgres"}'
# → {"key": "project/myapp/stack", "stored": true}

# Get
curl -X POST http://host:8766/tools/memory-get \
  -d '{"key": "project/myapp/stack"}'
# → {"found": true, "key": "...", "value": "Python, FastAPI, Postgres"}

# Search by prefix
curl -X POST http://host:8766/tools/memory-search \
  -d '{"prefix": "project/"}'
# → {"prefix": "project/", "count": 3, "results": [...]}

# Delete
curl -X POST http://host:8766/tools/memory-delete \
  -d '{"key": "project/myapp/stack"}'
# → {"deleted": "project/myapp/stack", ...}
```
