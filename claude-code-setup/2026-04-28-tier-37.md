# Tier 37: Task Management System

**Дата**: 2026-04-28
**Статус**: ✅ Реализовано
**Цель**: Добавить todo-трекинг — создание, обновление, список, удаление задач.

## Зачем

Claude Code умеет создавать todo-листы (TaskCreate) и отслеживать прогресс. Наш агент работал «вслепую» — без структурированного списка задач. Task Management замыкает цикл: агент может разбить сложную задачу на шаги, отслеживать прогресс и сообщать статус.

## Что сделано

### 1. Хранилище задач

**Файл**: `/root/deepseek-agent/agent.py`

```python
TASKS_FILE = os.path.join(_session_dir, "tasks.json")
_tasks = {}
if os.path.exists(TASKS_FILE):
```

### 2. 4 инструмента

| Инструмент | Параметры | Описание |
|-----------|-----------|----------|
| `task_create` | `subject` (req), `description` | Создать задачу, возвращает `task_id` |
| `task_update` | `task_id` (req), `status`, `subject`, `description` | Обновить статус/поля |
| `task_list` | `status` (all/pending/in_progress/completed) | Список задач с counts |
| `task_delete` | `task_id` (req) | Удалить задачу |

### 3. Формат задачи

```json
{
  "t73808695": {
    "id": "t73808695",
    "subject": "Implement login page",
    "description": "Add email/password form",
    "status": "in_progress"
  }
}
```

- ID: `t` + unix timestamp (8 последних цифр)

## API

```bash
# Create
curl -X POST http://host:8766/tools/task-create \
  -H 'Content-Type: application/json' \
  -d '{"subject": "Fix auth bug", "description": "..."}'
# → {"task_id": "t73808695", ...}

# Update
curl -X POST http://host:8766/tools/task-update \
  -H 'Content-Type: application/json' \
  -d '{"task_id": "t73808695", "status": "completed"}'

# List (all)
curl -X POST http://host:8766/tools/task-list \
  -H 'Content-Type: application/json' \
  -d '{}'

# List (filtered)
curl -X POST http://host:8766/tools/task-list \
  -H 'Content-Type: application/json' \
  -d '{"status": "in_progress"}'

# Delete
curl -X POST http://host:8766/tools/task-delete \
  -H 'Content-Type: application/json' \
  -d '{"task_id": "t73808695"}'
```

## Тестирование

```bash
# Create → OK
curl .../task-create ... → {"task_id": "t73808695", "status": "pending"}

# List → shows 1 pending task
curl .../task-list ... → {"total": 1, "summary": {"pending": 1, ...}}

# Update → completed
curl .../task-update ... → {"task_id": "t73808695", "status": "completed"}

# Delete → removed
curl .../task-delete ... → {"deleted": "t73808695", ...}

# List → empty
curl .../task-list ... → {"total": 0, "tasks": []}
```
