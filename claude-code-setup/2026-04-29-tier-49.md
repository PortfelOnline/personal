# Tier 49: DeepSeek Anthropic API proxy + Claude Code auto-stack

**Дата**: 2026-04-29
**Статус**: ✅ Реализовано + 5 токен-оптимизаций
**Цель**: Запускать Claude Code через DeepSeek Anthropic-совместимый API — одной командой `claude`, с автофиксом проблем (изображения, thinking).

## Проблемы DeepSeek Anthropic API

| # | Проблема | Решение |
|---|----------|---------|
| 1 | Нет vision — `type: "image"` блоки rejected | Прокси вырезает image/document блоки, заменяет на Groq-описание (llama-4-scout) или OCR (tesseract rus+eng) |
| 2 | `thinking.budget_tokens` — DeepSeek не поддерживает extended thinking | Прокси удаляет `thinking` блок целиком из запроса + фильтрует из ответа |
| 3 | Context window: DeepSeek 128K vs Claude 200K — может превысить лимит | Прокси считает точные токены (tiktoken cl100k_base), отрезает старые сообщения при превышении ~118K |
| 4 | **tool_result** — полный вывод `cat/grep/read` съедает 50-70% контекста | `_truncate_tool_results()` обрезает каждый блок до 2000 токенов |
| 5 | **cache_control** — DeepSeek не поддерживает Anthropic-кеширование | `_strip_cache_control()` удаляет из всех блоков |
| 6 | **Лишние пробелы/переносы** в текстовых блоках | `_normalize_text()`: 3+\n→2, rstrip per line, strip edges |

## Компоненты

### ds-proxy.py (`claude-code-setup/ds-proxy.py`)

HTTP-прокси на `localhost:8099`, перехватывает запросы Claude Code → DeepSeek:

```
Claude Code → localhost:8099 (ds-proxy) → api.deepseek.com/anthropic/v1/messages
```

- Вырезает `type: "image"` рекурсивно (в т.ч. внутри tool_result), заменяет на Groq-описание / OCR / placeholder
- Удаляет `thinking` блок целиком из запроса + фильтрует из ответа
- Точный подсчёт токенов через tiktoken cl100k_base, adaptive prompt budget ~118K
- **Двухфазное truncation**: Phase 1 — сбрасывает только tool-only сообщения, Phase 2 — oldest-first fallback
- **Условный max_tokens cap**: снижает только при переполнении, никогда < 4096 (учтён риск обрезания длинных ответов)
- **truncate tool_result**: обрезает вывод `cat/grep/read` до 2000 токенов/b
- **strip cache_control**: DeepSeek не поддерживает кеширование
- **Whitespace normalization**: сжимает переносы, чистит хвостовые пробелы (5-15% токенов)
- HEAD `/` endpoint для health-check Claude Code
- Поддерживает streaming (SSE passthrough) и non-streaming с корректным Content-Length
- Авто-подстановка `x-api-key` из `DEEPSEEK_API_KEY` env или `--api-key`

### .zshrc: `claude()` wrapper

Функция-обёртка, запускающая полный стек одной командой:

1. Проверяет, запущен ли прокси на :8099
2. Если нет — читает ключ из `~/.config/deepseek-api-key`, запускает `ds-proxy.py` фоново
3. Экспортирует `ANTHROPIC_BASE_URL=http://localhost:8099`
4. Маппинг моделей: Sonnet/Opus/Haiku → deepseek-v4-flash/pro/flash
5. Сбрасывает `ANTHROPIC_AUTH_TOKEN` (прокси подставляет ключ сам)
6. Запускает оригинальный `command claude`

### .claude/settings.json

`"model": "sonnet"` → `"model": "deepseek-v4-flash"`
