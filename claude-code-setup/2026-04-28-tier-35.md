# Tier 35: Git Commit & Push

**Дата**: 2026-04-28
**Статус**: ✅ Реализовано
**Цель**: Добавить `git commit` и `git push` как инструменты DeepSeek Agent.

## Зачем

Агент умел читать git status/diff/log, но не мог самостоятельно коммитить и пушить изменения. Это замыкает цикл: агент редактирует → коммитит → пушит без ручного вмешательства.

## Что сделано

### 1. `tool_git_commit(params)`

```python
def tool_git_commit(params):
    repo = params.get("repo_path", "")
    msg = params.get("message", "")
    if not repo or not msg:
        return json.dumps({"error": "repo_path and message required"})
    if not os.path.isdir(os.path.join(repo, ".git")):
        return json.dumps({"error": "not a git repository"})
    # git add -A && git commit -m "message"
```

- `git add -A` — stagges all changes
- `shlex.quote` для безопасного экранирования message
- Timeout: 30s

### 3. Регистрация

| Место | Что добавлено |

### 4. API

- `POST /tools/git-commit` — body: `{"repo_path": "...", "message": "..."}`
- `POST /tools/git-push` — body: `{"repo_path": "...", "remote": "origin", "branch": "..."}`
