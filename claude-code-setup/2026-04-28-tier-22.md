# Tier 22: Web Tools — HTTP-инструменты для DeepSeek Agent

**Дата**: 2026-04-28
**Статус**: 🔜 В ПЛАНЕ
**Цель**: DeepSeek Agent может ходить в веб, как Claude Code.

## Зачем

Сейчас агент не может:
- Читать веб-страницы по URL
- Искать в Google/Яндексе
- Проверять API-эндпоинты

## Что сделать

### 1. Инструмент `web_fetch(url, selector)`
- HTTP GET с User-Agent, парсинг HTML → текст
- Опциональный CSS selector для целевого контента
- Таймаут 10-15 секунд

### 2. Инструмент `web_search(query)`
- DuckDuckGo без API-ключа (или платный SerpAPI/DataForSEO)
- Возврат топ-N результатов с заголовками и сниппетами

### 3. Безопасность
- Таймауты, ограничение размера ответа (50KB)
- Нет доступа к internal IP (127.0.0.1, 10.x, 172.x, 192.168.x)
- rate limit: не более 10 запросов в минуту
