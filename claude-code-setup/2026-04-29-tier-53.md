### 2. Sub-agent Auto-routing (deepseek-agent)

Автоматическое определение типа задачи и роутинг к специализированному sub-agent.

**6 профилей:**

| Профиль | Системный промпт | Триггеры (ключевые слова) |
|---------|------------------|---------------------------|
| `code_review` | Senior code reviewer | review, code review, найди ошибки, проверь код |
| `debug` | Debugger | debug, ошибка, error, bug, crash, traceback |
| `refactor` | Refactoring expert | refactor, рефакторинг, переписать, clean up |
| `research` | Technical researcher | research, исследуй, compare, analyze |
| `write_code` | Coding agent | напиши, implement, реализуй, add feature |
| `ops` | DevOps engineer | server, deploy, docker, nginx, config, ssh |

**HTTP endpoints:**
- `GET /auto-route` — статус + список профилей
- `POST /auto-route` — toggle (`{"enabled": true/false}`)
- `GET /agents` — список всех sub-agent (включая auto-routed)
- `GET /agents/<id>` — конкретный sub-agent
- `POST /agents/<id>/kill` — убить запущенный sub-agent
- `auto_route: false` в `/chat` — opt-out для конкретного запроса

**Изменённый файл на сервере n:**
- `/root/deepseek-agent/agent.py` — классификатор, 6 профилей, `_auto_route()`, HTTP endpoints

**Проверка:**
```bash
# Статус
curl http://localhost:8766/auto-route
# Выключить
curl -X POST http://localhost:8766/auto-route \
  -H 'Content-Type: application/json' -d '{"enabled":false}'
# Список sub-agent
curl http://localhost:8766/agents
# Chat с auto-routing (по умолчанию включён)
curl -X POST http://localhost:8766/chat \
  -H 'Content-Type: application/json' \
  -d '{"message":"review this code: def add(a,b): return a+b", "session_id":"test"}'
```

### 3. Local Test-agent (Docker Desktop)

Локальный запуск ds-proxy + Claude Code CLI через Docker Desktop на Mac.

**Новые файлы (personal/claude-code-setup/):**

| Файл | Назначение |
|------|------------|
| `Dockerfile.ds-proxy` | Python 3.10-slim + tiktoken + ds-proxy.py |
| `Dockerfile.claude` | Node.js 20-slim + Claude Code CLI |
| `docker-compose.local.yml` | Полный стек (ds-proxy:8099 + claude-code) |
| `.env.example` | Шаблон переменных окружения |

**Использование:**
```bash
cd personal/claude-code-setup
cp .env.example .env
# Заполнить DEEPSEEK_API_KEY в .env
open -a Docker  # запустить Docker Desktop
docker compose -f docker-compose.local.yml up -d
docker compose -f docker-compose.local.yml run claude-code -p "review this code"
```
