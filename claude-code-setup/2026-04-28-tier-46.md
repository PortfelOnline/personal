# Tier 46: Structured hooks — bash_shield + usage_tracker

**Дата**: 2026-04-28
**Статус**: ✅ Реализовано
**Цель**: Добавить структурированные хуки: PreBash (защита от опасных команд), PostEdit (логирование редактирований), PostTask (логирование завершённых чатов).

## Зачем

Без этих хуков агент не имел защиты от опасных bash-команд (полагался только на `check_dangerous_cmd()` в `safe_access.py`, который требовал permission gate — но в цепочке инструментов злоумышленник мог вызвать `bash` до подтверждения). Не было аудита редактирований файлов и статистики по сессиям.

## Архитектура

### Новые события в hook_loader.py

```
valid_events = {PreToolUse, PostToolUse, PreChat, PostChat, PostInit, PreBash, PostEdit, PostTask}
```

### bash_shield (PreBash)

```
tool_bash()
  ├── PreBash hook (bash_shield)
  │   └── 13 dangerous patterns → {"blocked": True, "error": "bash_shield blocked: ..."}
  └── check_dangerous_cmd() (pre-existing, permission gate)
```

13 паттернов: `rm -rf /`, `rm -rf /*`, `> /dev/sda`, `dd if=`, `mkfs.`, `kill -9 -1`, `killall -9`, `docker kill`, `docker rm -f`, `reboot`, `shutdown`, `poweroff`, `init 0`.

Выполняется ДО `check_dangerous_cmd()` — блокирует на уровне хука, без permission gate.

### usage_tracker (PostEdit + PostTask)

```
tool_edit_file()
  ├── success → PostEdit {path, old_len, new_len, status: "ok"} → edit_log.jsonl
  └── error   → PostEdit {path, old_len, new_len, status: "error"} → edit_log.jsonl

handle_chat()
  ├── return (no tool calls) → PostTask {result, tool_calls_count, session_id} → task_log.jsonl
  ├── return (tool calls)    → PostTask {result, tool_calls_count=iteration+1, session_id} → task_log.jsonl
  └── iteration limit        → PostTask {result: "Tool call limit reached", tool_calls_count=TOOL_CALL_MAX_ITER, session_id} → task_log.jsonl
```

Log-файлы: `/app/data/edit_log.jsonl` (append-only, JSONL), `/app/data/task_log.jsonl` (append-only, JSONL).

## Тестирование

```bash
# Dangerous command → blocked by bash_shield
curl -X POST http://host:8766/tools/bash \
  -d '{"command": "rm -rf /"}'
# → {"blocked": true, "error": "bash_shield blocked: would delete root filesystem"}

# Safe command → works
curl -X POST http://host:8766/tools/bash \
  -d '{"command": "echo hello"}'
# → {"result": "hello\n"}

# Edit → PostEdit logged
docker exec deepseek-agent cat /app/data/edit_log.jsonl
# → {"ts": 1777385447.88, "event": "edit", "path": "/tmp/test_hooks.txt", ...}

# Chat → PostTask logged
docker exec deepseek-agent cat /app/data/task_log.jsonl
# → {"ts": 1777385470.24, "event": "task", "tool_calls_count": 1, ...}
```
